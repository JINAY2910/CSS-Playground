# CSS Playground 🎨

This repository is a structured collection of **CSS examples**, organized from beginner to advanced levels, along with a **handwritten PDF of notes** covering all topics.  
It is designed to help you learn CSS step by step with hands-on practice. 💻✨

## 📚 Levels Overview

1. **Level 1 - Basics** 🌱  
   - Introduction, syntax, colors, fonts, text, and box model.

2. **Level 2 - Selectors** 🧩  
   - Classes, IDs, combinators, pseudo-classes.

3. **Level 3 - Layout** 📐  
   - Flexbox, Grid, positioning, responsive design.

4. **Level 4 - Advanced** ⚡  
   - Transitions, transforms, and advanced properties.

5. **Level 5 - Animations** 🎬  
   - Keyframes, motion, and interactive effects.
